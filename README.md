Usage
---------

#### From the terminal (the window you want people to see)

	vim __out.js

#### set vim to autoload every 20 milliseconds

	:set nolz | while 1 | e | redraw | $ | sleep 20m | endw

#### In another terminal prompt

	node parse.js
